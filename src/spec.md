# Specification

## Summary
**Goal:** Add food photos to catalog cards in the Global Recipe application.

**Planned changes:**
- Add optional imageUrl field to FoodItem type in backend
- Update FoodCard component to display food photos above the food name
- Add image URL input field to the UploadRecipeForm component
- Generate and store food photos as static assets in frontend/public/assets/generated/

**User-visible outcome:** Users will see appetizing photos of each food item displayed on the catalog cards, making the browsing experience more visually engaging.
